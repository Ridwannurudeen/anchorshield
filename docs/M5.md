# M5 Hardening & Trust

## Scope

M5 adds:

- `docs/CEREMONY.md`
- `docs/SECURITY_REVIEW.md`
- `.github/workflows/ci.yml`
- `scripts/m5-verify.ps1`
- `AI_USAGE.md`
- RWA root-rotation regression test

## Verification

Local command:

```bash
npm run m5:verify
```

The command runs:

- JavaScript syntax checks
- JSON checks
- `npm audit --audit-level=high`
- M1/M2 circuit smoke generation and negative tests
- M4 disclosure verification
- M4 event indexing
- converter golden tests
- payment gate tests
- RWA gate tests

## CI

`.github/workflows/ci.yml` runs on push and pull request:

- Node install via `.node-version`
- Rust `1.96.0` + `wasm32v1-none`
- Circom `v2.2.3`
- `npm ci`
- JavaScript/JSON/audit checks
- circuit smoke
- disclosure/indexer scripts
- converter tests
- payment and RWA gate tests
